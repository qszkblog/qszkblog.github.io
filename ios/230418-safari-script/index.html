iOS Safari浏览器脚本插件：去广告、网页增强，让Safari更好用

2023.04.18

Safari 脚本插件：让 iPhone 浏览器也能装"扩展"

用过 Chrome 浏览器的朋友都知道插件有多好用——去广告、网页净化、视频增强等等。但 iOS 上的 Safari 原生不支持 Chrome 式扩展，需要借助专门的脚本工具来实现。

本文介绍一款 iOS Safari 脚本管理工具，安装后可以为 Safari 添加各种实用脚本功能。

主要功能

功能 说明
网页去广告 自动屏蔽网页广告、弹窗
视频增强 网页视频播放增强
网页净化 去除冗余元素，专注阅读
自定义脚本 支持导入自定义 JavaScript 脚本
全站生效 可设置为所有网站自动生效

安装教程

第一步：下载安装

从 App Store 下载安装应用。

第二步：激活扩展

安装后需要手动启用 Safari 扩展：

  1. 打开 iPhone 设置 → Safari 浏览器 → 扩展
  2. 找到已安装的脚本工具，打开开关
  3. 在权限设置中选择"所有网站都允许"

第三步：选择脚本

激活完成后打开 APP，选择你需要的脚本。重要：在权限设置中选择"所有网站都允许"，这样脚本才能自动生效。

常见问题

Q：安全吗？ A：通过 App Store 审核上架，脚本权限由 iOS 系统管控，安全性有保障。

Q：会影响 Safari 速度吗？ A：脚本体积很小，对浏览速度基本无影响。如果感觉变慢，可以在 APP 中关闭不需要的脚本。

Q：支持哪些脚本？ A：应用内置脚本库，涵盖去广告、视频增强、网页美化等多种类型。也支持导入自定义脚本。

Q：需要每次手动开启吗？ A：不需要。设置"所有网站都允许"后，脚本会自动在 Safari 中生效，无需手动操作。
