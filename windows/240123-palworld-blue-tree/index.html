幻兽帕鲁发光蓝树：技能果实获取指南

在《幻兽帕鲁》中，发光蓝树是一种散布在地图各处的特殊树木，每棵树可以掉落 3 个技能果实。这些果实可以让你的帕鲁立即学会新技能，不受元素类型限制。

技能果实有什么用？

  • 让帕鲁立即学习新技能，无需升级
  • 不受帕鲁元素类型限制，任意帕鲁都可使用
  • 部分高级技能只能通过果实获得

发光蓝树位置一览

1. 群岛区域（约10棵蓝树）

从小海湾航点出发，游过水面后爬上高原左转。群岛是技能树最密集的区域，主要产出电系和冰系技能果实。

2. 小型定居点西侧

到达小定居点后，向西侧爬山，在山顶可以找到被红色树叶包围的技能果树。

3. 沼泽岛

传送到地图远东侧的航点后爬山，找到一个隐藏洞穴，跳进去即可获得技能果实。

4. 黑曜石火山

传送到废墟要塞航点后向南行进，爬上高塔从高原向西寻找。这个区域导航难度较高，建议做好准备再前往。

如何使用技能果实？

  1. 接近发光蓝树，按 E 键拾取技能果实
  2. 打开背包，右键点击技能果实
  3. 从列表中选择要学习技能的帕鲁
  4. 确认后帕鲁立即学会该技能

提示：建议在 40 级左右再去收集技能果实，部分高级果实需要到达特定区域才能获取。
