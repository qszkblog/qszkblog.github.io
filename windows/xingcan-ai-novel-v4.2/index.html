星灿AI写小说助手 v4.2 AI自动写小说

2026.04.16

简介

星灿AI小说写作助手是专为网络小说创作者设计的智能工具，集成AI辅助创作、小说管理、章节编辑等全流程功能，助力作者高效创作。

核心功能

  • 书架管理：支持多部小说管理及TXT导出
  • 章节编辑：实时字数统计、自动保存
  • AI续写：智能续写章节内容
  • 角色设定：支持AI生成角色人设
  • 类型配置：小说类型模板配置
  • 故事梗概：AI一键生成故事梗概
  • 多AI模型兼容：支持多种AI服务API设置

AI接入说明

支持的AI服务

  • OpenAI（gpt3.5turbo、gpt4等）
  • 智谱AI（GLM4等）
  • 百度千帆（ERNIEBot等）
  • 其他支持OpenAI Chat Completions格式的AI服务

配置步骤

  1. 启动软件，进入 设置→AI接口设置
  2. 填写配置信息：
    • API Key：AI服务密钥（sk开头字符串）
    • 接口地址：OpenAI格式的API地址
    • 模型名称：如gpt3.5turbo、glm4、erniebot等
    • 最大生成长度：建议500-2000
    • 温度：建议0.8（平衡创意与连贯性）
  3. 点击保存配置

测试验证

  • 进入小说编辑界面
  • 点击AI生成开头或AI续写按钮
  • 若AI正常响应，说明接入成功

版本信息

  • 软件版本：v4.2 API增强版
  • 支持接口：所有OpenAI格式接口
  • 操作系统：Windows
  • AI功能需联网使用
