win10蓝屏分析工具-BlueScreenView中文版！

2023.01.07

前言

电脑蓝屏太常见了，电脑蓝屏了却不知道是什么问题，也不知道该怎么办，老是蓝屏严重影响工作效率。

无论是对于开发者还是对于普通使用者来说，其实想知道引起蓝屏的原因，方法很简单，今天就给大家推荐一个快速检测蓝屏问题的方法，那就是利用BlueScreenView这款来进行自动诊断。

介绍

该软件运行后会自动扫描机器上的蓝屏文件，并列出显示所有崩溃转储的文件，你可以通过展示出来的信息直观的看到蓝屏的软件信息，接下来就看下有哪些信息吧。

  1. 列出所有崩溃转储文件，包括崩溃转储日期/时间和崩溃细节。
  2. BlueScreenView枚举崩溃堆栈中的内存地址，并找到其中可能导致崩溃的驱动/模块。
  3. 可以模拟Windows崩溃时显示的蓝色屏幕进行信息显示。
  4. BlueScreenView自动定位崩溃转储中出现的驱动，并提取它们的版本资源信息，包括产品名称，文件版本，公司和文件描述。
  5. 只要您在高级选项里选择正确的转储文件夹，BlueScreenView也可让您查看另一个Windows中的崩溃数据。

使用方法也很简单，双击BlueScreenView.exe，运行蓝屏检测程序，运行后会自动扫描您的当前的存储器转储文件文件夹并列出显示所有崩溃转储文件，包括崩溃转储日期、时间和崩溃细节。

小结

本次分享的蓝屏日志分析程序，可以很只管的看出来是哪些模块导致的蓝屏，能够让你精准的进行排查和修复。
