常见问题

问：播放4K视频卡怎么办？

检查三点：

  1. 显卡是否支持DXVA（2015年后的显卡基本都支持）
  2. DXVA选项是否开启（设置→内部滤镜→视频解码）
  3. 视频编码是否为HEVC/H.265（这是最吃性能的格式）

如果显卡太老不支持DXVA，只能靠CPU软解，老电脑确实会卡。

问：为什么有些视频没声音？

可能是DTS/AC3音频编码问题。尝试：

  1. 设置→内部滤镜→音频解码→启用所有选项
  2. 或单独安装AC3Filter

问：MPC-HC和MPC-BE选哪个？

MPC-BE（Black Edition）是分支版本，界面更现代，功能略多。

  • 追求极致轻量 → MPC-HC
  • 想要更多功能 → MPC-BE

两者都是开源免费，核心体验差不多。

问：能播放蓝光原盘吗？

可以。MPC-HC支持BDMV文件夹结构，但需要正确配置解码器。网上搜"MPC-HC 蓝光"有详细教程。
