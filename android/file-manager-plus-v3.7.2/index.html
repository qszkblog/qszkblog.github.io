前言

智能电视的功能越来越丰富，可以安装各种应用程序、存储和播放多媒体文件。在这种情况下，一个高效的文件管理工具就显得尤为重要。本文介绍一款简洁、响应速度快、操作体验好的电视端文件管理工具——文件管理器+。

软件简介

文件管理器+ 是一款方便实用的文件管理器，支持电视和手机双平台，完美适配遥控操作。可以管理电视内存、云存储等，并支持局域网、WebDAV、网盘以及 SMB3.0 和 SMB2.0 等协议。
